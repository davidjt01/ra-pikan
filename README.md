# RA-PIKAN: Residual-Adaptive Physics-Informed Kolmogorov–Arnold Networks

Implementation of **RA-PIKAN**, a hybrid three-cycle training algorithm that interleaves
physics-residual diagnosis with spatially targeted B-spline grid densification
and residual-adaptive collocation augmentation (RAR-D, Wu et al. 2023).

The RA-PIKAN algorithm runs:
- **Cycles 0–1**: targeted B-spline extension, concentrating new knots in high-residual sub-domains
- **Cycle 2**: uniform grid extension across the full domain (avoids ill-conditioned least-squares projection after two targeted extensions)
- **Probe-gated training**: 30% of each cycle's budget is a probe; if improvement < 10%, remaining steps are deferred to a supplementary Adam phase after all cycles

## Project structure

| Path | Purpose |
|------|---------|
| `src/` | Library: KAN/PIKAN layers, PDE residuals, benchmarks, RA-PIKAN algorithm, trainers, metrics, symbolic extraction |
| `experiments/` | Scripts to run benchmarks and generate figures/tables |
| `results/` | CSV outputs, convergence logs, PDF figures (generated by scripts) |
| `requirements.txt` | Pinned Python dependencies |

## Setup

```bash
# Python 3.9+ required
python -m venv .venv

# Windows
.venv\Scripts\activate
# Linux / macOS
source .venv/bin/activate

pip install -r requirements.txt
```

## Running experiments

### Quick smoke test

```bash
python experiments/run_all.py --quick --device cpu
```

### Full experiment suite

Runs all 4 benchmarks × 6 methods × 5 seeds on CPU:

```bash
python experiments/run_all.py --device cpu
```

Already-present results are skipped automatically; the run is safe to interrupt
and resume.

### Run individual benchmarks

```bash
# 1D Burgers (ν=0.01, seed=42, all methods)
python experiments/run_burgers.py --nu 0.01 --seeds 42 --device cpu

# 1D Allen-Cahn (default 2000-step budget)
python experiments/run_allen_cahn.py --eps 0.1 --seeds 42 --device cpu

# 2D Advection-Diffusion
python experiments/run_advection_diffusion.py --Pe 100 --seeds 42 --device cpu

# 1D Duffing (symbolic extraction sub-study)
python experiments/run_duffing.py --seeds 42 --device cpu
```

### Generate all figures and tables

```bash
python experiments/make_figures_and_tables.py
```

Outputs written to `results/`:
- `table_accuracy.csv` — main L² accuracy table (mean ± std per method per benchmark/param)
- `table_compute.csv` — compute cost (elapsed, steps/s, peak memory)
- `table_symbolic.csv` — symbolic extraction formulas and R² values (Duffing)
- `fig_bar_burgers.pdf`, `fig_bar_allen_cahn.pdf`, `fig_bar_adv_diff.pdf` — bar charts of final L² error by method
- `fig_convergence_burgers_nu{nu}.pdf`, `fig_convergence_ac_eps{eps}.pdf`, `fig_convergence_ad_Pe{Pe}.pdf` — training loss vs. Adam iteration
- `fig_l2cycle_burgers_nu{nu}.pdf`, `fig_l2cycle_ac_eps{eps}.pdf`, `fig_l2cycle_ad_Pe{Pe}.pdf` — L² error vs. refinement cycle
- `fig_solution_burgers_nu{nu}.pdf`, `fig_solution_ac_eps{eps}.pdf`, `fig_solution_adv_diff_Pe{Pe}.pdf` — predicted vs. reference solution fields
- `fig_residuals_burgers_nu{nu}_{method}.pdf`, `fig_residuals_adv_diff_Pe{Pe}_{method}.pdf`, `fig_residuals_ac_eps{eps}_{method}.pdf` — spatial PDE residual maps per cycle
- `fig_gini_evolution.pdf` — Gini coefficient evolution across refinement cycles
- `fig_kan_activations_duffing.pdf` — KAN edge activation curves (Duffing, seed 42)
- `fig_duffing_symbolic.pdf` — symbolic extraction R² distribution (Duffing)
- `fig_sensitivity_percentile.pdf` — RA-PIKAN sensitivity to residual percentile threshold

## Methods

| Method | Key file | Description |
|--------|----------|-------------|
| MLP-PINN | `src/models.py` | Tanh MLP — [2,64,64,64,1] for Burgers/Allen-Cahn/Adv-Diff, [1,64,64,1] for Duffing |
| Fixed ChebyPIKAN | `src/models.py` | Chebyshev KAN — [2,10,10,1] / [1,10,10,1], degree 5, no grid refinement |
| Fixed B-spline PIKAN | `src/models.py` | B-spline KAN — same widths, grid=5, no grid extension (ablation baseline) |
| RAR-D PINN | `experiments/run_*.py` | MLP + residual-adaptive collocation augmentation (Wu et al. 2023) |
| Uniform-ext PIKAN | `src/ra_pikan.py` | B-spline KAN + global uniform grid extension each cycle |
| **RA-PIKAN** | `src/ra_pikan.py` | B-spline KAN + 2 targeted cycles + 1 uniform cycle + probe-gated training |

## Benchmarks

| Benchmark | Parameters | Seeds | Challenge |
|-----------|-----------|-------|-----------|
| 1D Burgers | ν ∈ {0.01, 0.005, 0.001} | n=5 | Sharp shock front |
| 1D Allen–Cahn | ε ∈ {0.1, 0.05} | n=5 | Stiff phase interface |
| 2D Advection–Diffusion | Pe ∈ {100, 500} | n=5 | Exponential boundary layer |
| 1D Duffing | δ=0.5, α=1.0, β=0.1, γ=0.3, ω=1.2 | n=5 | Nonlinear ODE; symbolic extraction |

Seeds: 42, 123, 456, 789, 1234 for all benchmarks.

## Step budget (equal across all methods, per Wu et al. 2023)

| Benchmark | Adam steps | L-BFGS steps | RA-PIKAN / Uniform-PIKAN phase split |
|-----------|-----------|-------------|---------------------------------------|
| Burgers / Allen-Cahn | 2000 | 100 | 500 base + 3×500 refinement + 100 L-BFGS |
| Adv-Diff | 3000 | 200 | 750 base + 3×750 refinement + 200 L-BFGS |
| Duffing (B-spline methods) | 3000 | 200 | 750 base + 3×750 refinement + 200 L-BFGS |
| Duffing (RA-PIKAN, ChebyKAN) | 3000 | 200 | 1000 base + 2×1000 refinement + 200 L-BFGS† |

†Duffing RA-PIKAN uses ChebyKAN (no B-spline grid extension) with 2 refinement cycles; the symbolic extraction sub-study targets the Chebyshev basis.

## Key results (5 seeds, mean L² relative error)

### Allen-Cahn (ε=0.1 and ε=0.05)

| Method | ε=0.1 | ε=0.05 |
|--------|-------|--------|
| MLP-PINN | 8.71e-02 | 1.22e-01 |
| RAD-PINN | 9.69e-02 | 1.34e-01 |
| Fixed B-spline PIKAN | 1.39e-02 | 1.14e-02 |
| Fixed ChebyPIKAN | 6.22e-03 | 1.01e-02 |
| Uniform-ext PIKAN | 6.85e-04 | 1.23e-03 |
| **RA-PIKAN** | **4.99e-04** | **4.75e-04** |

RA-PIKAN beats Uniform-ext PIKAN by **1.37×** (ε=0.1) and **2.58×** (ε=0.05).

### Advection-Diffusion

| Method | Pe=100 | Pe=500 |
|--------|--------|--------|
| MLP-PINN | 3.69e+00 | 4.97e+00 |
| RAD-PINN | 3.03e+00 | 3.07e+00 |
| Fixed B-spline PIKAN | 3.11e+00 | 3.11e+00 |
| Uniform-ext PIKAN | 2.98e+00 | 3.08e+00 |
| **RA-PIKAN** | **1.76e+00** | 3.09e+00 |

RA-PIKAN is best for Pe=100 (1.69× over Uniform-ext). At Pe=500 all methods produce L²≈3 — the boundary layer (~0.002) is below collocation resolution for all methods.

### Burgers

All methods yield high L² errors at low viscosity due to shock sharpness. Fixed ChebyPIKAN outperforms RA-PIKAN here (global Chebyshev basis better suited than local B-spline refinement for sub-grid shock fronts).

| Method | ν=0.01 | ν=0.005 | ν=0.001 |
|--------|--------|---------|---------|
| Fixed ChebyPIKAN | **5.25e-02** | **1.49e-01** | 3.39e-01 |
| MLP-PINN | 1.34e-01 | 2.85e-01 | 3.63e-01 |
| RAD-PINN | 1.35e-01 | 2.71e-01 | 3.51e-01 |
| RA-PIKAN | 1.84e-01 | 3.18e-01 | 3.55e-01 |
| Uniform-ext PIKAN | 2.21e-01 | 2.88e-01 | **2.69e-01** |
| Fixed B-spline PIKAN | 3.02e-01 | 3.19e-01 | 3.52e-01 |

## Reproducibility

- All scripts fix `torch.manual_seed` and `numpy.random.default_rng` at the start.
- Hyperparameters are defined as constants / dataclass fields at the top of each script.
- Results follow the naming convention `{benchmark}_{method}_{param}_seed{seed}.csv`.
- `make_figures_and_tables.py` reads only from `results/` — no re-running needed.
